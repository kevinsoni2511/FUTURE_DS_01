# ​ Power BI Project – Task 1: Business Sales Dashboard

As part of my **Data Science & Analytics – Future Interns** journey, I developed an **interactive Power BI dashboard** using the `swiggy_cleaned.csv` dataset to analyze restaurant performance, customer preferences, and pricing trends on Swiggy.

##  Project Highlights

**🔢 Key KPIs (Cards)**  
- **Average Price:** ₹348.44  
- **Average Rating:** 3.66  
- **Total Restaurants:** 7,865  

**📊 Visuals & Insights**  
- **🏙 Top Cities by Restaurant Count** – Identify high-density markets  
- **🍜 Top 5 Most Common Cuisines** – Highlighting customer food preferences  
- **⏱ Delivery Time Distribution** – Peak delivery window insights  
- **💰 Price vs Rating** – Cost vs satisfaction analysis  
- **⭐ Average Rating per City** – Comparative satisfaction across cities  
- **🗺 Restaurant Map** – Geographical spread across cities  

---

##  Technical Stack & Implementation

- **Power Query** – Data cleaning, standardization, and preparation  
- **DAX** – Created a `FoodTypes` table using `GENERATE`, `PATHITEM` for multi-cuisine handling  
- **Modeling** – Linked `FoodTypes[ID] ↔ swiggy_cleaned[ID]` with bidirectional filtering for seamless interactions  
- **Visual Design** – Built using bar charts, pie charts, scatter plots, maps, KPI cards with cohesive styling  
- **Interactive Filters** – Slicers for **City**, **Price Range**, **Rating Range** for dynamic exploration  

---

##  Key Learning Outcomes

- Built a **normalized model** for better dashboard performance  
- Mastered **cross-filtering** to make visuals interactive  
- Transformed complex columns (multi-valued cuisines) into analysis-ready tables with **DAX**  
- Delivered a dashboard optimized for clarity, interactivity, and effective business storytelling
